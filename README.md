## üß† The Gemini 3 Integration
Web Gen is built to showcase the specific 2026 features of the Gemini 3 ecosystem.

* **Agentic Reasoning:** We utilize the `thinking_level: HIGH` parameter. This forces the model to reason through DOM hierarchy and CSS layout logic before generating code, drastically reducing syntax errors.
* **High-Context Memory:** Leveraging Gemini 3‚Äôs **1M+ context window**, Web Gen stores the entire project history, allowing for iterative design (e.g., "Now add a contact form that matches the header's style") with perfect architectural consistency.
* **Low-Latency Synthesis:** By optimizing our calls for **Gemini 3 Flash**, we achieve near-instantaneous code generation, keeping the developer in a "flow state."

## ‚ú® Key Features

## üöÄ Getting Started
### **Requirements**
* Android 5.0 (Lollipop) or higher.
* An active Gemini API Key (configured in the app settings).
* English (UK) system language recommended.

### **Installation**
1.  Clone this repository:
    ```bash
    git clone [https://github.com/LuminousOrganisation/Web-Gen.git](https://github.com/LuminousOrganisation/Web-Gen.git)
    ```
2.  Open the project in **Android Studio**.
3.  Build and run the app on your physical device or emulator.
